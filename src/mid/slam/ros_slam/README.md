# Install dependencies
- **gmapping_slam** 

  for ubuntu 16.04:

```
sudo apt-get install ros-kinetic-gmapping
```

​       for ubuntu 18.04:

```
sudo apt-get install ros-melodic-gmapping
```

- cartographer_slam
  for ubuntu 16.04:

```
sudo apt-get install ros-kinetic-cartographer*
```

​       for ubuntu 18.04:

```
sudo apt-get install ros-melodic-cartographer*
```
